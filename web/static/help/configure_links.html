About Mattermost

Mattermost is a team communication service. It brings team real-time messaging and file sharing into one place, with easy archiving and search, accessible across PCs and phones.

We built Mattermost to help teams focus on what matters most to them. It works for us, we hope it works for you too. Learn more, or download the source code from http://mattermost.com.

How to update this link

In the source code, search for "config.js" and update the links pointing to this page to whatever policies and product description you prefer.
